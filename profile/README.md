## Public projects

### [gmacFTP](https://github.com/GMAC-pl/gmacftp)

A fast, secure, native **FTP / FTPS / SFTP client for macOS**, built in Rust.

- Rust core, native macOS interface
- FTP, FTPS and SFTP support
- macOS Keychain-backed secret storage
- Signed and notarized macOS release
- Development preview, already usable for everyday transfer work

**Download:** [latest gmacFTP release](https://github.com/GMAC-pl/gmacftp/releases)
